h1. Отчёт details

h2. Описание

Если указан один счёт, то отчёт выводит все проводки по счёту за указанный период. Если указана группа счетов, то выводятся все проводки по каждому из счетов. В результате, одна проводка может быть упомянута в отчёте несколько раз (в списках проводок разных счетов).

h3. Параметры

Один необязательный параметр — путь к счёту или группе счетов. Если не указан, отчёт строится по всем счетам.
