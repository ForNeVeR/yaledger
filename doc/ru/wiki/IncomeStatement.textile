h1. Отчёт incomestatement — доходно-расходная ведомость

h2. Описание

Отчёт выводит две части дерева счетов:
* Доходы (income). Счетами доходов считаются все дебетные счета.
* Расходы (outcome). Счетами расходов считаются все кредитные счета.

Для каждого из этих счетов выводится текущий баланс, по группам счетов подводятся итоги. Кроме того, подводится общее сальдо по доходам-расходам.

h2. Параметры

Один необязательный параметр — путь к группе счетов, в которой искать счета расходов/доходов. Если не указан, отчёт строится по всем счетам.

Одна опция @-z@, @--no-zeros@. Если указана, то счета с нулевым балансом не выводятся.
