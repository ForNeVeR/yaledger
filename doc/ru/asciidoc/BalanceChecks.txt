Проверка балансов счетов
========================

Перед выполнением полупроводки YaLedger может проверить, достаточно ли средств на счёте для этого. Такие проверки выполняются только для тех счетов, для которых настроено.

Эти настройки определяются параметрами счёта info, warning, error:

    account A1 {
      info = 1000;
      warning = 100;
      error = 0;
    };

Суммы в параметрах указываются в валюте счёта. Перед выполнением каждой полупроводки по такому счёту выполняются проверки:

* Если после полупроводки баланс будет меньше чем info — выдаёт сообщение INFO (сообщение будет видно только с ключом командной строки +-d info+).
* Меньше чем warning — выдаёт WARNING.
* Меньше чем error — выдаёт ERROR и завершает работу.

Если один из этих параметров не указан, соответствующая проверка не выполняется.

Для кредитовых счетов вместо проверки «баланс меньше указанного» выполняется проверка «баланс больше указанного».
